A multi-scale deep learning pipeline using a lightweight U-Net architecture to detect and segment localized flood extents from Sentinel-1 SAR imagery. The framework implements a "trained broad, validated narrow" approach—pretraining on global baseline datasets (Sen1Floods11), fine-tuning on regional Philippine event layers (HDX/PhilSA), and validating strictly against high-resolution local holdouts (Aringay, La Union). Features an integrated geospatial preprocessing track and a secondary binary classification module for ground-level crowd-sourced community photos.
